**현재 강점:**

- ✅ 간단하고 직관적
- ✅ 오프라인 작동
- ✅ 층간소음 ZERO, 책상 앞 리프레시 등 특화 루틴

**마케팅 메시지:**

## 🚀 성장 로드맵

### Phase 1: 기반 구축 (1-2주)

#### A. ASO (앱스토어 최적화) - 최우선!
**현재:**
```
앱 이름: "Set1"
검색 키워드: 약함
```

**개선안:**
```
앱 이름: "Set1 - 맨몸운동 홈트레이닝"
또는: "Set1 홈트: 기구없이 집에서 운동"

키워드: 홈트, 맨몸운동, 층간소음, 책상운동,
       체중운동, 루틴, 초보자
```

**스크린샷 개선:**
- 현재: 단순 화면 캡처
- 개선: 핵심 기능 강조
  - "층간소음 ZERO 하체 루틴"
  - "책상에서 10분 리프레시"
  - "운동 기록이 한눈에"

#### B. 초기 사용자 확보

**1. 지인 10명 (1시간)**
```
- 다운로드 + 리뷰 요청
- 별점 4.5 이상 목표
- 리뷰 5개 이상 → 앱스토어 노출 개선
```

**2. 커뮤니티 홍보 (주 2-3회)**

**타겟 커뮤니티:**
- 클리앙 (운동/헬스 게시판)
- 뽐뿌 (다이어트/헬스)
- 디시인사이드 (헬스갤)
- Reddit (r/bodyweightfitness)
- Facebook 그룹 (홈트레이닝)

**홍보 전략:**
```
❌ "제가 만든 앱입니다"
✅ "층간소음 없이 홈트하는 방법 [후기+앱 소개]"
✅ "사무실에서 10분 운동 루틴 [영상+앱]"
```

**3. SNS 해시태그**
```
#홈트 #맨몸운동 #층간소음zero #책상운동
#홈트레이닝 #체중운동 #운동루틴
```

### Phase 2: 기능 개선 (2-4주)

#### 우선순위 높음 (사용자 50명 달성 후)

**1. 히트맵 캘린더** (개발 2일)
```
GitHub 스타일 운동 기록 시각화
⬜ 🟩 🟩 ⬜ 🟩 🟩 🟩

- 시각적 임팩트
- 스트릭 동기부여
- SNS 공유 욕구
```
**예상 효과:** 리텐션 +30%, SNS 공유 증가

**2. 운동 설명 추가** (개발 3일)
```typescript
{
  name: "일반 푸시업",
  videoUrl: "링크" // 또는 GIF
  description: "1. 어깨너비로 손을 짚고...",
  commonMistakes: ["엉덩이가 올라감"],
  tips: ["코어에 힘을 주세요"]
}
```
**예상 효과:** 초보자 이탈률 -40%

**3. 간단한 성취 시스템** (개발 2일)
```
🥉 첫 운동 완료
🥈 7일 연속 운동
🥇 30일 연속 운동
💪 푸시업 100개 달성
```
**예상 효과:** 동기부여 강화, 리텐션 +25%

#### 우선순위 중간 (사용자 100명 달성 후)

**4. 부위별 밸런스 레이더 차트** (개발 1일)
```
맨몸운동 특화 → 부위별 밸런스 중요
약점 파악 → 루틴 추천 연결
```

**5. 월간 리포트** (개발 3일)
```

🔥 최장 스트릭: 5일

[이미지로 공유하기]
```
**예상 효과:** SNS 바이럴 가능성

**6. 진행 사진 비교** (개발 2일)
```

체중, 운동량 변화 비교
```
**예상 효과:** 동기부여 최강

## ⚠️ 중요한 인사이트

### 통계 강화 vs 마케팅

**현재 상황:**
- 사용자 5명
- 통계 기능 이미 기본은 갖춤

**딜레마:**
```
통계 2주 개발 → 멋진 기능 완성
하지만 여전히 5명 사용자
→ 빛 좋은 개살구
```

**권장 방향:**
```
1. 히트맵 캘린더만 빠르게 구현 (2일)
2. 스크린샷 업데이트
3. 마케팅 집중 (2주)
4. 50명 달성 후 추가 통계 개발
```

### 솔직한 자기 평가: "내가 이 앱을 쓸까?"

**현재 상태:** 아니요 😔

**이유:**
1. 이미 삼성헬스 사용 중
2. 전환 동기 없음
3. 운동 설명/영상 없음

**쓸 것 같은 시나리오:**
- ✅ 층간소음 고민 → "층간소음 ZERO" 루틴 매력적
- ✅ 사무직 몸 뻐근함 → "책상 앞 리프레쉬" 필요성 공감
- ✅ 헬스 초보 → "초보자 홈트 입문" 진입장벽 낮음

**개선 방향:**
1. **명확한 포지셔닝** - 특정 니즈 공략
2. **운동 설명 추가** - 초보자 친화
3. **동기부여 요소** - 성취감, SNS 공유

## 🤔 전략적 결정 필요

### 버전 업데이트 (1.0.9 vs 1.0.10)

**현재 상황:**
- 1.0.9 심사 대기 중 (운동 46개, 루틴 10개 추가)
- 1.0.10 준비됨 (체중 입력 개선, 키보드 버그픽스, 리프레시 루틴)

**권장:**
- ✅ **1.0.9 그대로 진행**
- 5명 사용자에게는 버전 차이 무의미
- 심사 시간에 마케팅하는게 더 중요
- 1.0.10은 사용자 증가 후 제출

### 개발 vs 마케팅 시간 배분

**현재 문제:**
- 개발 90%, 마케팅 10%
- → 완성도는 높지만 사용자 없음

**권장 배분 (향후 1개월):**
- 개발 30% (히트맵 캘린더, 버그픽스)
