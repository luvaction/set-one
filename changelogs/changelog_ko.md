# 변경 사항 (Changelog)

## 앱스토어 상시 정보 (버전 무관)

### 프로모션 텍스트 (최대 170자)

원탭으로 끝나는 운동 기록! 복잡한 입력은 그만, 운동에만 집중하세요. 광고 없음, 완전 무료. 지금 바로 Set1과 함께 운동을 시작해보세요!

### 앱 설명

Set1은 운동 기록이 가장 쉬운 앱입니다.

다른 앱은 세트 하나 기록하는데 5번 탭해야 하지만, Set1은 버튼 하나면 끝입니다.

주요 기능:
- 원탭 기록: 세트 완료를 한 번의 탭으로
- 스마트 입력: 이전 무게와 횟수 자동 입력
- 의미있는 통계: 진짜 필요한 데이터만 표시
- 루틴 관리: 나만의 운동 루틴 저장 및 실행
- 운동 기록: 모든 운동 기록 자동 저장
- 성장 추적: 개인 기록(PR)과 주간 비교
- 완전 무료: 광고 없음, 구독 없음
- 오프라인 지원: 인터넷 없이도 모든 기능 사용 가능

운동에만 집중하세요. 기록은 Set1이 알아서 합니다.

## 버전 1.0.12

### ✨ 새로운 기능

- **운동별 최대 중량 추이 차트**
  - 선택한 운동들의 최대 중량 변화를 시간순으로 추적
  - 여러 운동을 동시에 비교 가능
  - 일/주/월/연별 기간 필터링 지원

- **운동 일수 차트**
  - 주/월/년 단위로 실제 운동한 일수를 막대 그래프로 시각화
  - 운동 꾸준함을 한눈에 확인

- **개인 기록 (PR) 섹션**
  - 운동별 최대 중량, 최대 횟수, 최고 볼륨 기록
  - 상위 5개 운동을 랭킹 형태로 표시

### 🎨 UI/UX 개선

- **통계 화면 대폭 개선**
  - 핵심 지표 카드 3열로 통합 (연속 운동, 주간 목표, 올해 활동)
  - 실용적인 데이터 중심으로 차트 재구성
  - 불필요한 복잡한 UI 제거

## 버전 1.0.11

### ✨ 새로운 기능

- **볼륨 추이 차트 추가**
  - 운동별 볼륨(무게×횟수×세트) 추이를 한눈에 확인
  - 여러 운동을 동시에 비교 가능
  - 일/주/월/연별 기간 필터링 지원
  - 차트 클릭 시 최대 중량, 평균 반복수 등 상세 정보 표시

- **Imperial 단위 시스템 지원**
  - 프로필에서 Metric(kg, cm) 또는 Imperial(lb, ft/in) 선택 가능
  - 모든 무게 및 키 데이터가 선택한 단위로 자동 변환
  - 운동 기록, 통계, 히스토리 화면 모두 지원

### 🎨 UI/UX 개선

- **통계 화면 개편**
  - 실제로 유용한 지표만 남기고 불필요한 섹션 제거
  - 총 중량 누적값 표시 제거 (의미 없는 수치)
  - 개인 기록(PR), 인사이트, 운동 유형 분포 섹션 제거
  - 볼륨 추이 차트로 진짜 중요한 발전 상황 확인 가능

- **기록 화면 간소화**
  - 전체 누적 통계 제거 (총 운동 횟수, 총 운동 시간)
  - 캘린더와 실제 기록에 집중

### 🐛 버그 수정

- 단위 중복 표시 문제 해결
  - Imperial 설정 시 "79 lb lb" 처럼 단위가 중복 표시되던 문제 수정
  - 기록 화면에서 "70 kgkg" 중복 표시 문제 수정

- 운동 시간 계산 개선
  - 휴식 시간이 포함되어 비현실적으로 높게 표시되던 문제 수정
  - 실제 운동 시간만 정확하게 집계하도록 개선

- 번역 키 누락 수정
  - 운동 세션에서 무게/체중 입력 시 번역 키가 그대로 표시되던 문제 해결
  - 근육 부위 번역 누락 문제 해결 (목, 팔)

### 🌐 다국어 지원

- 새로운 번역 키 추가
  - 볼륨 추이, 운동 선택 등 신규 기능 관련 번역
  - 근육 부위 번역 완성도 개선
