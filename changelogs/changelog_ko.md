# 변경 사항 (Changelog)

## 앱스토어 상시 정보 (버전 무관)

### 프로모션 텍스트 (최대 170자)

원탭으로 끝나는 운동 기록! 복잡한 입력은 그만, 운동에만 집중하세요. 광고 없음, 완전 무료. 지금 바로 Set1과 함께 운동을 시작해보세요!

### 앱 설명

Set1은 운동 기록이 가장 쉬운 앱입니다.

다른 앱은 세트 하나 기록하는데 5번 탭해야 하지만, Set1은 버튼 하나면 끝입니다.

주요 기능:
- 원탭 기록: 세트 완료를 한 번의 탭으로
- 스마트 입력: 이전 무게와 횟수 자동 입력
- 의미있는 통계: 진짜 필요한 데이터만 표시
- 루틴 관리: 나만의 운동 루틴 저장 및 실행
- 운동 기록: 모든 운동 기록 자동 저장
- 성장 추적: 개인 기록(PR)과 주간 비교
- 완전 무료: 광고 없음, 구독 없음
- 오프라인 지원: 인터넷 없이도 모든 기능 사용 가능

운동에만 집중하세요. 기록은 Set1이 알아서 합니다.

## 버전 1.0.11

### ✨ 새로운 기능

- Imperial 단위 시스템 지원
  - 프로필에서 Metric(kg, cm) 또는 Imperial(lb, ft/in) 선택 가능
  - 모든 무게 및 키 데이터가 선택한 단위로 자동 변환
  - 운동 기록, 통계, 히스토리 화면 모두 지원

### 🐛 버그 수정

- 근육 부위 번역 누락 문제 해결 (목, 팔)
  - "책상 앞 리프레쉬 10분" 루틴에서 근육 부위가 번역 키 그대로 노출되던 문제 수정
  - 모든 근육 부위가 한국어/영어로 올바르게 표시되도록 개선

### 🌐 다국어 지원

- 근육 부위 번역 완성도 개선
  - `muscleGroups.neck` (목/Neck) 번역 키 추가
  - 한국어 및 영어 근육 부위 매핑 강화

### 🔧 기술 개선

- 번역 헬퍼 함수 개선
  - 영어 값이 직접 입력될 경우를 대비한 매핑 추가
  - 번역 일관성 향상

## 앱스토어 제출용 (v1.0.11)

### 업데이트 내역

이번 업데이트에서는 Imperial 단위 시스템 지원과 다국어 지원을 개선했습니다.

새로운 기능:

- Imperial 단위 시스템 지원: 프로필에서 Metric 또는 Imperial 단위를 선택할 수 있습니다

개선 사항:

- 모든 운동 부위가 선택한 언어로 올바르게 표시됩니다
- 루틴 화면에서 번역 누락 문제를 해결했습니다
- 더욱 매끄러운 다국어 사용 경험을 제공합니다

항상 Set1을 사용해주셔서 감사합니다!
